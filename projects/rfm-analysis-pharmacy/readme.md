# RFM-анализ клиентской базы аптечной сети

## Цель проекта

В условиях сезонного снижения выручки аптечная сеть планирует использовать SMS-рассылки для стимулирования повторных покупок и увеличения выручки. Для этого необходимо сегментировать клиентов по их покупательскому поведению, чтобы персонализировать коммуникации и повысить их эффективность.

## Цели исследования

- Выделить клиентские сегменты по поведению покупателей
- Определить, какие сегменты приносят основную выручку
- Предложить практические рекомендации по удержанию и реактивации клиентов

## Методология и ключевые результаты

### 1. RFM-сегментация
На основе данных о транзакциях клиентам были присвоены баллы по трем метрикам: давность, частота и денежная ценность. В результате база была разделена на 5 стратегических сегментов.

### 2. ABC-анализ
Для оценки концентрации выручки и распределения маркетинговых усилий клиенты были разделены на три класса по накопительной доле выручки: 
* Класс **A** (первые 80% выручки)
* Класс **B** (следующие 15%)
* Класс **C** (оставшиеся 5%)

Методы RFM и ABC были объединены в матрицу для поиска пересечений и зон риска, что позволило дать более точечные рекомендации.

## Ключевые инсайты по сегментам

*   **Ядро бизнеса (86,5% выручки):** Сегменты «Чемпионы» и «Лояльные» вместе составляют 52,3% клиентской базы, но генерируют подавляющую долю всей выручки (54% + 32,5%). Это основная зона для инвестиций в лояльность.
*   **Подтверждение принципа Парето:** 35,3% клиентов (класс A) обеспечивают 80% выручки компании. Удержание каждого клиента из этой группы критически важно.
*   **Критическая зона риска (Класс A):** Обнаружены 84 клиента из класса A, которые находятся в сегменте «Под угрозой». Потеря даже 30% из них нанесет ощутимый удар по итоговой выручке. Требуются персональные коммуникации.
*   **Зона неиспользованного потенциала:** 25 клиентов из «Чемпионов» находятся в классе B. Это идеальная целевая группа для апселлинга, так как они уже активны и лояльны.
*   **Зона экономии маркетингового бюджета:** Сегмент «Потерянные» класса C насчитывает 953 клиента. Вложения в привлечение/удержание этой группы экономически нецелесообразны. Для них следует ограничиться низкобюджетными массовыми коммуникациями.

## Итоговые стратегические рекомендации для бизнеса

На основе совмещенной матрицы RFM + ABC сформирована стратегия дифференцированного подхода. Не все клиенты требуют одинакового уровня внимания — ценные сегменты должны получать персонализированные предложения, а менее активные — экономичные и автоматизированные действия.

| Сегмент | ABC-класс | Приоритет | Рекомендация |
| :--- | :--- | :--- | :--- |
| **Чемпионы** | A | Удержание | Персональные условия, VIP-предложения, бонусы за повторные покупки. |
| **Лояльные** | A/B | Развитие | Кросс-продажи и программы для роста среднего чека. |
| **Под угрозой** | A/B | Реактивация | Целевые бонусы и персонализированные предложения для возврата активности. |
| **Нельзя терять** | A | Приоритетный win-back | Индивидуальные коммуникации и специальные условия для предотвращения потери выручки. |
| **Потерянные** | B/C | Тестирование | Низкобюджетные и осторожные кампании с оценкой эффективности. |
