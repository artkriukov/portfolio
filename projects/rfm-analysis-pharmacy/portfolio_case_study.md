# RFM-анализ клиентской базы аптечной сети

## Краткое описание проекта

Этот проект демонстрирует, как можно использовать RFM-анализ для сегментации клиентов аптечной сети и выбора целевых коммуникаций для роста выручки. В отличие от простого описания данных, здесь показан полный цикл аналитики: от бизнес-задачи и подготовки данных до сегментации, оценки вклада клиентов в выручку и формулирования практических рекомендаций.

## Бизнес-задача

В условиях сезонного снижения выручки компания хочет не просто привлекать новых клиентов, а работать с существующей базой — возвращать в активные покупки тех, кто уже покупал раньше, и удерживать наиболее ценных клиентов. Для этого необходимо:

- выделить клиентские сегменты по поведению;
- оценить, какие сегменты обеспечивают основную выручку;
- определить, какие группы клиентов нуждаются в удержании, а какие — в реактивации.

## Что было сделано

1. Подготовлена и очищена транзакционная база по бонусным картам.
2. Выполнен RFM-анализ по трем признакам:
   - Recency — давность последней покупки;
   - Frequency — частота покупок;
   - Monetary — сумма выручки.
3. Выполнен ABC-анализ по вкладу клиентов в общую выручку.
4. Сформированы рекомендации по управлению клиентскими сегментами.

## Методология

Для анализа использовалась таблица транзакций по бонусным картам. В исследование включались только те покупки, где клиент мог быть идентифицирован по карте. Это позволило оценить поведение клиента на уровне истории покупок, а не только по разовым транзакциям.

### RFM-подход

RFM-модель делит клиентов по трем измерениям:

- Recency — как давно клиент делал последнюю покупку;
- Frequency — как часто клиент покупает;
- Monetary — сколько денег клиент приносит компании.

На основе распределения по квартилям каждому клиенту присваиваются баллы по каждому признаку. Затем формируется итоговый сегмент.

## Основные выводы

- В базе присутствуют как активные и лояльные клиенты, так и группы с высоким риском оттока.
- Не все клиенты одинаково важны для бизнеса: часть клиентов приносит заметно большую долю выручки, чем остальные.
- Для роста выручки важно не только привлекать новых клиентов, но и грамотно работать с уже существующей аудиторией.

## Вклад ABC-классов в общую выручку

ABC-анализ позволяет определить, какая часть клиентов приносит основную выручку. Это особенно важно для бизнеса, потому что коммуникации и бонусные программы должны быть направлены не на всех одинаково, а с учетом ценности клиента.

### Почему это важно

Если небольшая группа клиентов приносит основную часть выручки, то компания должна:

- удерживать этих клиентов более активно;
- использовать персональные предложения;
- не тратить одинаковый бюджет на всех без разбора.

### Код для анализа

```python
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns
from pathlib import Path
from dotenv import load_dotenv
from sqlalchemy import create_engine, text

load_dotenv()

DATABASE_URL = (
    f"postgresql+psycopg2://{os.getenv('DB_USER')}:"
    f"{os.getenv('DB_PASSWORD')}@"
    f"{os.getenv('DB_HOST')}:"
    f"{os.getenv('DB_PORT')}/"
    f"{os.getenv('DB_NAME')}"
)

engine = create_engine(DATABASE_URL)


def read_sql_from_file(filename):
    query = Path(f"sql/{filename}.sql").read_text(encoding="utf-8")
    return pd.read_sql(text(query), engine)


rfm_df = read_sql_from_file("rfm_segmentation")
abc_rfm_df = read_sql_from_file("abc_revenue")

abc_rfm_df = abc_rfm_df[[
    "card", "segment", "abc_revenue", "total_revenue", "cum_revenue_pct"
]]

abc_summary = (
    abc_rfm_df.groupby("abc_revenue")
    .agg(
        customers=("card", "count"),
        revenue=("total_revenue", "sum")
    )
    .reset_index()
)

abc_summary["revenue_share_pct"] = (
    abc_summary["revenue"] / abc_summary["revenue"].sum() * 100
)
abc_summary["customer_share_pct"] = (
    abc_summary["customers"] / abc_summary["customers"].sum() * 100
)

plt.figure(figsize=(8, 4))
sns.barplot(data=abc_summary, x="abc_revenue", y="revenue_share_pct", palette="Blues")
plt.title("Вклад ABC-классов в общую выручку")
plt.xlabel("ABC-класс")
plt.ylabel("Доля выручки, %")
plt.tight_layout()
plt.show()
```

### Как интерпретировать результат

Если класс A формирует основную часть выручки, то это означает, что бизнесу стоит сфокусироваться на защите и развитии этого сегмента. Визуально это выглядит как концентрация выручки в небольшом количестве клиентов. Такая картина характерна для компаний с высокой ценностью ключевых клиентов.

### Дополнительный анализ: распределение по сегментам и ABC-классам

```python
segment_abc = (
    abc_rfm_df.groupby(["segment", "abc_revenue"])
    .agg(customers=("card", "count"))
    .reset_index()
)

pivot = (
    segment_abc.pivot(index="segment", columns="abc_revenue", values="customers")
    .fillna(0)
)

plt.figure(figsize=(8, 4))
sns.heatmap(pivot, annot=True, fmt=".0f", cmap="Blues")
plt.title("Распределение клиентов по RFM-сегментам и ABC-классам")
plt.tight_layout()
plt.show()
```

Этот шаг помогает увидеть, какие сегменты содержат наиболее ценных клиентов. Например, сегмент с высокой активностью и высокой выручкой может стать приоритетом для персональных предложений, а менее ценные группы — для более экономичных коммуникаций.

## Практические рекомендации

1. Для сегмента "Чемпионы" и "Лояльные" использовать персональные условия, бонусы за повторные покупки и программы повышения среднего чека.
2. Для сегментов с высоким риском оттока использовать целевые win-back-кампании, напоминания о покупке и персонализированные предложения.
3. Для менее ценных клиентов использовать более осторожный и бюджетный подход, чтобы не расходовать маркетинговый ресурс без отдачи.
